# Vishal Motor Training School Website

A modern, professional landing page for Vishal Motor Training School, Kandivali East, Mumbai.

## Features
- **Responsive Design**: Works perfectly on Mobile, Tablet, and Desktop.
- **Micro-animations**: Interactive scroll-reveal effects for a premium feel.
- **WhatsApp Integration**: Direct chat button and form redirection.
- **Glassmorphism UI**: Modern frosted-glass aesthetics.

## How to Run the Project

Since this is a static website, you can run it in two ways:

### 1. Simple Open (Recommended for Quick View)
Double-click the `index.html` file in your project folder. It will open in your default web browser (Chrome, Edge, etc.).

### 2. Using a Development Server (Recommended for Developers)
If you have VS Code, you can use the **Live Server** extension:
1. Open the folder in VS Code.
2. Right-click `index.html`.
3. Select **"Open with Live Server"**.

Alternatively, you can use `npx` if you have Node.js installed:
```powershell
npx serve .
```

## Project Structure
- `index.html`: Main page structure.
- `style.css`: Custom premium styling and animations.
- `script.js`: Interactivity and logic.
- `assets/`: Place your `logo.png` here.
